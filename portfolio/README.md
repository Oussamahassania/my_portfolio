# Oussama Hassania - Portfolio

A modern, responsive portfolio website showcasing my work as a Fullstack & Backend Developer.

## 🚀 Features

- **Modern Design**: Black & Red Marble theme with glass morphism effects
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: Framer Motion animations for a dynamic user experience
- **Interactive Sections**: 
  - Hero section with tech stack showcase (icons only, no text)
  - About section with personal information
  - CV download section
  - Projects showcase with filtering
  - Contact form with social media links
- **Tech Stack Icons**: Visual representation of technologies used (Java, Spring Boot, React, Angular, Bootstrap, GitHub)
- **Contact Integration**: Direct links to email, LinkedIn, and GitHub

## 🛠️ Technologies Used

- **React** - Frontend framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling framework
- **Framer Motion** - Animation library
- **React Icons** - Icon library

## 📦 Installation

1. Navigate to the portfolio directory:
```bash
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## 🎨 Customization

### Personal Information
Update the following files with your information:
- `src/components/Hero.jsx` - Name and tech stack
- `src/components/About.jsx` - Personal description and photo
- `src/components/Contact.jsx` - Contact information
- `src/components/Projects.jsx` - Your projects

### Styling
- Main colors are defined in `tailwind.config.js`
- Black & Red Marble theme can be customized in the CSS variables
- Glass morphism effects can be adjusted in the `.glass` class

### Adding Your CV
1. Add your CV file to the `public` folder as `cv.pdf`
2. The download functionality is already implemented in `src/components/CVSection.jsx`

## 📱 Sections
